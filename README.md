# AtlasLIBYA - متجر إلكتروني ليبي

## نظرة عامة

AtlasLY هو متجر إلكتروني حديث ومصمم خصيصاً للسوق الليبي، متخصص في بيع اللابتوبات وأجهزة الكمبيوتر والإكسسوارات الأصلية. تم تطويره باستخدام Django مع دعم كامل للغة العربية والتوجه من اليمين إلى اليسار (RTL).

## المميزات الرئيسية

### 🏪 الواجهة العامة
- **الصفحة الرئيسية**: عرض أحدث المنتجات والعروض الخاصة في شرائط تفاعلية
- **بحث وفلترة متقدمة**: البحث بالاسم والوصف والعلامة التجارية مع فلاتر التصنيف والسعر
- **عرض المنتجات**: تصميم بطاقات حديث مع معلومات شاملة ودعم للصور المتعددة
- **صفحة المنتج**: تخطيط ثلاثي الأعمدة (معرض الصور - المعلومات - صندوق الشراء)
- **المفضلة**: نظام إضافة وإدارة المنتجات المفضلة مع أيقونة القلب التفاعلية

### 🛒 نظام التسوق والطلبات
- **السلة**: تصميم عصري مشابه لـ Amazon مع إدارة الكميات وحساب الإجماليات
- **الدفع**: صفحة دفع موحدة مع الدفع عند الاستلام (COD)
- **الشراء الفوري**: إمكانية الشراء المباشر بنقرة واحدة
- **تتبع الطلبات**: عرض تفاصيل الطلبات وحالة التسليم

### 👤 إدارة الحسابات
- **تسجيل الدخول والحساب**: نظام شامل لإدارة المستخدمين
- **صفحة الحساب الشخصي**: عرض تاريخ الطلبات والمفضلة والبيانات الشخصية
- **المفضلة**: إدارة كاملة للمنتجات المفضلة مع إحصائيات

### ⚙️ لوحة الإدارة
- **إدارة المنتجات**: إضافة وتعديل المنتجات مع دعم الصور المتعددة
- **التصنيفات الذكية**: إنشاء تلقائي للتصنيفات عند كتابة اسم جديد
- **إدارة الطلبات**: متابعة وتحديث حالة الطلبات
- **العروض والخصومات**: نظام مرن للعروض الخاصة والخصومات

## التقنيات المستخدمة

- **Backend**: Django 5.2+, Python 3.12+
- **Database**: MySQL
- **Frontend**: Bootstrap 5 RTL, Bootstrap Icons
- **Styling**: CSS3 مخصص مع متغيرات وتحريكات
- **JavaScript**: Vanilla JS للتفاعلات والـ AJAX

## طريقة التثبيت والتشغيل

### 1. استنساخ المشروع
```bash
git clone https://github.com/sulima2901/AtlasLIBYA.git
cd AtlasLIBYA
```

### 2. إعداد البيئة الافتراضية
```bash
python -m venv atlas_env
source atlas_env/bin/activate  # على Linux/Mac
# أو
atlas_env\Scripts\activate  # على Windows
```

### 3. تثبيت المتطلبات
```bash
pip install Django Pillow pymysql
```

### 4. إعداد قاعدة البيانات
```bash
# إنشاء قاعدة بيانات MySQL جديدة
mysql -u root -p
CREATE DATABASE atlasly_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# تحديث إعدادات قاعدة البيانات في atlasly/settings.py
```

### 5. تنفيذ الترحيلات
```bash
python manage.py makemigrations
python manage.py migrate
```

### 6. إنشاء مستخدم إداري
```bash
python manage.py createsuperuser
```

### 7. تشغيل الخادم
```bash
python manage.py runserver
```

الآن يمكنك زيارة الموقع على `http://127.0.0.1:8000`

## الميزات المتقدمة

### نظام المفضلة
- إضافة وإزالة المنتجات بتقنية AJAX
- عداد المفضلة في شريط التنقل
- صفحة منفصلة لإدارة المفضلة

### البحث والفلترة
- بحث متقدم في الأسماء والأوصاف والعلامات التجارية
- فلاتر متعددة: التصنيف، العلامة التجارية، نطاق السعر
- شيبس فلاتر قابلة للإزالة
- ترتيب حسب التاريخ والاسم والسعر

### نظام الطلبات المرن
- دعم الطلبات للمستخدمين المسجلين وغير المسجلين
- معلومات شاملة للعميل والتسليم
- تتبع حالة الطلب (قيد المعالجة، مدفوع، مشحون، تم التسليم)
- نظام الدفع عند الاستلام (COD)

### الشرائط التفاعلية (Carousels)
- عرض أحدث المنتجات في شريط متحرك RTL
- عرض العروض الخاصة
- تحريك تلقائي كل 6 ثوانِ مع إيقاف عند التمرير
- أزرار تنقل يدوي
